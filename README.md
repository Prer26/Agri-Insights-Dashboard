## 📊 Project Overview

This project is a **Power BI dashboard** built to analyze crop production patterns across India.

The goal was to transform raw agricultural data into meaningful insights using **data visualization and storytelling techniques**.

⏱️ Built as a **30-minute challenge (extended to 35 mins)**

---

## 🎯 Objectives

- Analyze crop production across states and districts  
- Identify dominant crops and regional patterns  
- Explore relationships between area and production  
- Build an interactive and user-friendly dashboard  

---

## 📌 Key Insights

✨ A small number of crops dominate total production  
📍 Strong regional concentration across specific states  
📈 Significant variation in production and cultivated area  

---

## 📊 Features

- KPI Cards (Total Crops, Districts, Production, Area)  
- Interactive filters (Crop, State, District)  
- Category-wise production analysis  
- Area distribution visualization  
- Geo-map for regional insights  

---

## 🚀 How to Use

1. Download the `.pbix` file  
2. Open in Power BI Desktop  
3. Interact using filters and slicers  
4. Explore insights
